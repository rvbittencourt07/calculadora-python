# Calculadora Simples (Python)

Uma calculadora de linha de comando que avalia expressões automaticamente.

## Como executar

No PowerShell (na pasta do arquivo):

```powershell
python "calculadora.py"
```

## Exemplos

- `2*(3+4)`
- `10/3`
- `2**8`
- `sqrt(16) + 5`

## Saida

Digite `sair` ou `quit` para encerrar.
